 работа: Учёт личных расходов

 Цель

Реализовать консольное приложение для учёта личных расходов.
---

 Описание

Необходимо разработать программу expenses.py, которая позволяет пользователю записывать свои расходы, разбивать их по категориям и анализировать.

Программа работает через аргументы командной строки.

---
 Основные команды

1. Добавить расход

python expenses.py add <стоимость> <категория> <название>
---

 2. Добавить категорию

python expenses.py add-category <категория>
Добавляет новую категорию.
---

 3. Показать все расходы

python expenses.py list [категория]
Выводит список всех расходов.
Опциональный аргумент, при указываниии которого будет выводиться только траты по категории.
---

 4. Показать сумму расходов

python expenses.py total  [категория]
Выводит сумму всех расходов.
Опциональный аргумент, при указываниии которого будет выводиться сумма всех расходов в категории.


 Хранение данных
Формат хранения на выбор студента
можно, например CSV или JSON

---

 Требования

данные сохраняются между запусками программы
проверка существования категории при добавлении расхода
корректная обработка ошибок (неверный ввод)
использование стандартных библиотек Python
